# Модуль №8 Продвинутый React

Это проект [Next.js](https://nextjs.org), созданный с помощью [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Приступая к работе (Знакомство с Next.js)

Сначала запустите сервер разработки:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Чтобы увидеть результат, откройте в браузере [http://localhost:3000](http://localhost:3000).

Вы можете начать редактирование страницы, внеся изменения в app/page.tsxфайл. Страница автоматически обновляется по мере редактирования файла.

В этом проекте используется [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) автоматическая оптимизация и загрузка [Geist](https://vercel.com/font), нового семейства шрифтов для Vercel.
